Frequently Asked Questions

Are generated passwords saved anywhere?+
No. Passwords are generated using your browser's built-in crypto API and are never stored, logged, or transmitted.
How strong is a 16 character password?+
A 16-character password with mixed characters has trillions of possible combinations and would take centuries to crack by brute force.
Should I use symbols in passwords?+
Yes if the site allows it. Symbols dramatically increase password strength. If a site does not allow symbols use a longer password instead.
How often should I change passwords?+
Change passwords immediately if a site is breached. Otherwise use a unique strong password for each account rather than rotating frequently.
